# California Property Analysis with Excel

This project analyzes California property data using Microsoft Excel. It focuses on cleaning, organizing, and visualizing data to provide insights into property prices, features, and trends.

## Project Overview
The goal of this project is to leverage Excel's tools to clean and analyze the dataset, create meaningful visualizations, and extract actionable insights about California's real estate market.

## Dataset
- **Source:** Single-family property dataset  
- **Records:** 9,582  
- **Columns include:** Price, area (sqft), bedrooms, bathrooms, year built, property type, and property description.

## Data Cleaning & Preparation
The dataset was cleaned in Excel to ensure reliable analysis:

1. **Handling Missing Values**  
   - Used filters to identify missing or incomplete data  
   - Removed rows with critical missing fields (price, area)  
   - Filled minor missing data using average or median values  

2. **Data Formatting**  
   - Converted numerical columns to number format  
   - Standardized text in categorical columns such as property type  

3. **Removing Duplicates**  
   - Excel’s Remove Duplicates tool was used to avoid repeated records  

4. **Outlier Detection**  
   - Created conditional formatting rules to identify extreme values in price and area  
   - Filtered unrealistic values to focus on valid property listings  

5. **Data Sorting & Filtering**  
   - Sorted data by price, area, and number of bedrooms/bathrooms  
   - Used filters to explore specific property categories  

## Analysis & Visualizations
- Generated histograms and bar charts for price distribution and property features  
- Created scatter plots to explore relationships between area, price, and number of bedrooms  
- Used pivot tables to summarize data by property type, number of bedrooms, and year built  
- Built dynamic charts for easy trend visualization
